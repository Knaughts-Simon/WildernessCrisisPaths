# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-26

- Added trampoline ascension perks allowing Wilderness empires to take Galactic Nemesis and Cosmogenesis crisis paths.
- Requires Nemesis DLC (for Become the Crisis) and/or Machine Age DLC (for Cosmogenesis), plus Biogenesis DLC (for Wilderness origin).
- Compatible with other mods — does not override any vanilla files.
